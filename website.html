The HTML Section Heading elements :

The h1 to h6 tags are used to define HTML headings.

h1 defines the most important heading. h6 defines the least important heading.

HTML Code:


<h1 >Yousef </h1 >

<h2 >Yousef </h2 >

<h3 >Yousef </h3 >

<h4 >Yousef </h4 >

<h5 >Yousef </h5 >

<h6 >Yousef </h6 >

Output:


Yousef

Yousef

Yousef

Yousef

Yousef
Yousef

The br produces a line break in text (carriage-return). It is useful for writing a poem or an address, where the division of lines is significant.

HTML Code:


<p >Yousef </p >

<p >Yousef </p >




<p >Yousef </p >

<br >

<p >Yousef </p >

The Paragraph element :

The p represents a paragraph. Paragraphs are usually represented in visual media as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation, but HTML paragraphs can be any structural grouping of related content, such as images or form fields.

HTML Code:


<p > This is the first paragraph of text.

This is the first paragraph of text.

This is the first paragraph of text.</p >



<p > This is the second paragraph of text.

This is the second paragraph of text.

This is the second paragraph of text.</p >

Output:


This is the first paragraph of text. This is the first paragraph of text. This is the first paragraph of text.





This is the second paragraph of text. This is the second paragraph of text. This is the second paragraph of text.

The Anchor element

he <a> (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. Content within each <a> should indicate the link's destination. If the href attribute is present, pressing the enter key while focused on the <a> element will activate it.

HTML Code:


<a href="https://www.google.com/?hl=ar"> Google </a >

<a href="https://www.youtube.com/"> YouTube </a >

<a href="https://www.facebook.com/"> Facebook </a >

Output:


Google

YouTube

Facebook
